## gametype
- B : 모든적 부수기(Destroy all enemies)
- F : 기지 방어 하기(Defend your base)
- H : 상자 전부 줍기(Pick up all boxes)
- T : 모든 벙커 부수기(Destroy all bunkers)

## items
- X : 붉은 별(점수)
- L : 삽(기지 주변 벽 재생성)
- Y : 적 회복
- R : 총탄(일시적으로 장전 속도 증가)
- M : 메디킷(체력 회복)
- Z : 자물쇠(적 공격 일시 잠금)
- C : 시계(적 움직임 정지)
- B : 폭탄(적 전체 데미지)
- S : 방패(일시적 적 공격 무효화)
- I : 지뢰(밟으면 데미지)

## weather
- NONE : 맑음
- RAIN : 비
- SNOW : 눈
